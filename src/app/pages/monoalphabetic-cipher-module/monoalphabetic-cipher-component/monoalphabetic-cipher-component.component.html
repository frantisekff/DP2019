Description of cipher

Monoalphabetic cipher is a substitution cipher, which the key is randomly arranged alphabet. For example, if 'B' is encrypted as 'L', for any number of occurrence in that plaintext, 'B' will always get encrypted to 'L'.

Find decryption key with Hill Climbing alghorithm

Monoalphabetic cipher has to much combinations of key. We can't use a brute force attack, but meta-heuristic algorithm can be useful.

We use Hill Climbing alghorithm, which consists of these steps:
  • swap 2 letters of key and compute freguencies of bigrams
  • calculate difference between calculated frequencies of bigrams and referal values for English alphabet
  • save only better result
  • if we got better result then in next step we use this key from better result.

These steps we repeat for n times/number iterations.

You can set the number of iterations in the tab below and the number of attempts. The number of attempts represents which times this whole Hill climbing will run.
