Caesar Cipher Brute force attack

Caesar Cipher is replace every letter in message with diferent letter. We try find a shift, with text analysis. Then we use brute force attack to find decrypted message. We use index of coincidence and decryted text freqencies to evaluate the correctness of decrypted message. You can set up message and shift.

Simple Substitution cipher Hill climbing attack

Simple Substitution cipher substituting every plaintext character for a different ciphertext character. It differs from the Caesar cipher in that the cipher alphabet it is completely jumbled. We try find a key, with Hill climbing alghorithm. You can set up configuration of alghorithm and then see results.

Vigenere cipher Statistic analysis attack

Vigenere Cipher using a series of Caesar ciphers, based on the letters of a keyword. We try find a key, with text analysis. We use index of coincidence and freqencies of letters to find a key.
