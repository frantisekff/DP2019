Caesar cipher Brute force attack

Ceasar cipher replaces every letter in a message with a different letter. You can set up a message and shift. At first step of our attack on this cipher, we try to find the shift by the text analysis. Then we make the brute force attack to find a decrypted message. We use the index of coincidence and frequencies of decrypted text to evaluate the correctness of the decrypted message.

Simple Substitution cipher Hill climbing attack

Simple Substitution cipher substitutes every plaintext character for a different ciphertext character. It differs from the Ceasar cipher is that the cipher alphabet/key is completely jumbled. We try to find a key, with the Hill climbing algorithm. You can set up a configuration of the algorithm and then see results.

Vigenere cipher Statistic analysis attack

Vigenere cipher uses a series of Caesar ciphers, based on the letters of a keyword. We try to find a key with the text analysis. We use the index of coincidence and frequencies of letters.
