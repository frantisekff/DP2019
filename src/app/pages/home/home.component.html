Caesar Cipher Brute force attack

Caesar Cipher is replacing every letter in a message with a different letter. We try to find a shift, with text analysis. Then we use a brute force attack to find a decrypted message. We use the index of coincidence and frequencies of decrypted text to evaluate the correctness of the decrypted message. You can set up a message and shift.

Simple Substitution cipher Hill climbing attack

Simple Substitution cipher substituting every plaintext character for a different ciphertext character. From Caesar cipher differs in that the alphabet/key is completely jumbled. We try to find a key, with the Hill climbing algorithm. You can set up a configuration of the algorithm and then see results.

Vigenere cipher Statistic analysis attack

Vigenere Cipher is using a series of Caesar ciphers, based on the letters of a keyword. We try to find a key, with text analysis. We use the index of coincidence and frequencies of letters to find a key.
