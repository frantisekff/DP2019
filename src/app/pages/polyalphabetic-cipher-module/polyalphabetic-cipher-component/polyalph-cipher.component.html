Description of cipher
The Vigenere Cipher is a method of encrypting alphabetic text by using a series of Caesar ciphers, based on the letters of a keyword. In a Caesar cipher, each letter of the alphabet is shifted by several places.

For example, in a Caesar cipher with shift 3, „A“ would become „D“, „B“ would become „E“, „Y“ would become „B“ and so on. The Vigenere cipher uses several Caesar ciphers in sequence with different shift values.

Alghorithm to find length of key
We try find length of key with Friedman test.

Basically, Friedman test is when we separate text into boxes, number of boxes is length of key. For example when we have key of length 3 boxe, in first box will be first letter from encrypted text(CT), in second box second letter form CT, in third box third letter of CT. Then we repeat and in first box will be fourth letter of CT, in second box will be fifth letter and so on.

Then we compute the frequency of every box and Index Coincidence(IC). For boxes we compute the average of ICs. We repeat these steps and we compute average ICs for the key of length <2,16> In this example we expect that text is in the English language.

Description of attack
We try to decrypt message and find out the key of encrypted message. We use the guess of length from the previous step. We decrypt message for all combination and we calculate a score for this message. The score function is calculated as differences between frequencies of letters (decrypted message) and referral values of the English language. Finally, the score function sum these differences and this is the score for one key. We expect that the lowest score determines the key, which we are looking for. The frequencies of the correct message will be similar to referral values.
